# 🐍 Python 学习闯关平台

一个以闯关形式学习 Python 的全中文网站，特别针对软件测试方向设计。

## ✨ 特性

- 📚 **系统化学习路径**：从 Python 基础语法到软件测试框架，循序渐进
- 🎯 **闯关学习模式**：21 个精心设计的关卡，涵盖基础、工具、测试、项目四大类别
- 🔄 **自由练习**：所有题目都可以重复练习，没有通关门槛
- 📝 **错题本功能**：自动收集错题，方便复习巩固
- 🎨 **多样化题型**：选择题、填空题、编程题均匀分布
- 🛠️ **实战导向**：教授如何编写测试工具（数据迁移、测试数据生成等）
- 🏆 **终极项目**：搭建自己的自动化测试框架

## 🚀 快速开始

### 环境要求
- Python 3.7+
- pip

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/Jenwie-wj/python-learning.git
cd python-learning
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
python app.py
```

4. 打开浏览器访问
```
http://localhost:5000
```

## 📖 使用指南

### 基本操作

1. **设置用户名**：首次使用时，点击"更改用户名"按钮设置您的用户名，系统会记录您的学习进度

2. **选择关卡**：在"闯关学习"页面，点击任意关卡卡片查看题目列表

3. **答题**：点击题目进入答题界面
   - **选择题**：点击选项进行选择
   - **填空题**：在输入框中填写答案
   - **编程题**：在代码编辑区编写代码

4. **查看错题**：切换到"错题本"标签页，查看和复习做错的题目

5. **学习进度**：在"学习进度"页面查看整体完成情况

### 学习建议

- 💡 **循序渐进**：建议按照关卡顺序学习，每个关卡都为后续内容打基础
- 🔁 **重复练习**：做错的题目会自动加入错题本，建议定期复习
- 💻 **动手实践**：编程题建议在本地环境实际运行，加深理解
- 📚 **查阅文档**：遇到不懂的知识点，建议查阅 Python 官方文档
- 🎯 **完成项目**：认真完成实战项目，将知识融会贯通

## 🛠️ 技术栈

- **后端**：Flask + SQLAlchemy
- **前端**：HTML + CSS + JavaScript（原生）
- **数据库**：SQLite

## ⚠️ 安全说明

**重要**：本项目是一个学习平台演示项目，编程题的代码执行功能使用了简化实现。在生产环境中部署时，**必须**实现以下安全措施：

1. 使用沙箱环境执行用户代码（如 Docker 容器、RestrictedPython 等）
2. 限制代码执行时间和资源使用
3. 实施严格的输入验证和过滤
4. 添加用户认证和授权机制

当前实现仅适用于本地学习和测试环境。
